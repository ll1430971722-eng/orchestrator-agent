# jst-profit-report (alpha)

## 状态：🔵 Alpha

## 用途

利润简报：基于订单成本和售价的毛利估算。

## 触发条件

- 用户说"利润怎么样"、"毛利分析"、"赚钱了吗"

## 执行

基于订单数据 + 商品成本数据计算：
- 收入（订单实付金额）
- 成本（商品采购成本 × 销量）
- 毛利 = 收入 - 成本
- 毛利率 = 毛利 / 收入

## 输出模板

```markdown
# 利润简报 YYYY-MM-DD
## 概览
- 期间: YYYY-MM-DD ~ YYYY-MM-DD
- 总收入: ¥X
- 总成本: ¥Y
- 毛利: ¥Z（毛利率: XX%）

## 按渠道
| 渠道 | 收入 | 成本 | 毛利 | 毛利率 |
```
