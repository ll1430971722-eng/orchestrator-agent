# 抖音电商每日/每周/每月运营SOP

## 每日必做（30分钟）

| 时间 | 动作 | 工具/入口 |
|------|------|----------|
| 早9:00 | 查看昨日数据概览（成交额/订单/访客/退款） | 抖店首页 |
| 早9:05 | 查看体验分变化 | 抖店 → 店铺 → 商家体验分 |
| 早9:10 | 处理待发货订单 | 抖店 → 订单 → 订单管理 |
| 早9:15 | 处理售后（退款/退货/工单） | 抖店 → 售后 → 售后工作台 |
| 早9:20 | 回复评价（好评感谢，差评解释） | 抖店 → 商品 → 评价管理 |
| 早9:25 | 查看千川投放数据（如有投放） | 抖店 → 推广 → 千川 |
| 全天 | 客服在线（响应<30秒） | 飞鸽客服 |
| 全天 | 监控退款率，有异常及时排查 | 售后工作台 |

## 每周必做（2-3小时）

| 动作 | 说明 |
|------|------|
| 分析7天数据趋势 | GMV走势、转化漏斗变化、退款率趋势 |
| 优化商品标题 | 根据搜索热词变化更新 |
| A/B测试主图 | 每周测1-2个主推品 |
| 策划3-5条短视频 | 围绕主推品，不同角度 |
| 联系3-5个新达人 | 精选联盟找合作 |
| 复盘广告效果 | 关停ROI差的，加投好的 |
| 分析退款原因TOP5 | 针对性优化商品/详情页 |
| 检查库存 | 主推品备货充足 |

## 每月必做（半天）

| 动作 | 说明 |
|------|------|
| 月度数据复盘 | GMV/订单/客单价/退款率/体验分全盘 |
| 商品结构优化 | 淘汰滞销品，主推潜力品 |
| 竞品分析 | 同行在卖什么、什么打法、什么价格 |
| 内容策略调整 | 什么类型的视频效果好 |
| 达人合作复盘 | 哪些达人出单，加大合作 |
| 下月计划 | 目标、主推品、活动、预算 |
| 旺季提前布局 | 提前1个月准备大促/开学季 |

## 旺季前1个月特别SOP

| 倒计时 | 动作 |
|--------|------|
| T-30天 | 确定主推品，优化详情页+主图 |
| T-25天 | 备货，确保库存充足 |
| T-20天 | 制作旺季内容素材（短视频+直播脚本） |
| T-15天 | 联系达人预约档期 |
| T-10天 | 设置大促活动（限时秒杀/满减/优惠券） |
| T-7天 | 开启千川预热投放 |
| T-3天 | 检查所有设置，确保无误 |
| T-0 | 大促当天全员在线 |
