# 抖音电商算法机制

## 一、核心逻辑：兴趣电商的"货找人"

抖音电商本质是**兴趣电商**，核心是"货找人"而非"人找货"。

### 推荐机制三要素

| 要素 | 说明 | 如何优化 |
|------|------|---------|
| **用户标签** | 浏览/点赞/评论/停留时长/购买记录生成 | 内容要精准匹配目标用户画像 |
| **商品标签** | 由标题关键词、SKU主图、类目属性决定 | 标题写满30字，属性填全 |
| **匹配推荐** | 用户标签与商品标签高度契合时触发推荐 | 关键词精准 + 人群精准 |

### 2025年变化：内容场+货架场双轮驱动

- **内容场**（短视频/直播）：种草引流，激发兴趣
- **货架场**（搜索/商城/商品卡）：承载转化与复购
- 货架场GMV占比已达**40%**，搜索流量同比增长**210%**

## 二、商品卡入池机制（猜你喜欢）

### 入池条件
| 条件 | 标准 |
|------|------|
| 商品信息质量分 | ≥ 60分 |
| 历史销量 | 有成交记录 |
| 重复铺货 | 非重复铺货 |
| 店铺体验分 | ≥ 65分 |
| 商品状态 | 在售，非违规 |

### 不入池怎么查
抖店 → 商品 → 商品成长 → 更多运营工具 → 猜你喜欢准入

## 三、体验分（DSR）深度解析

### 评分构成
体验分 = f(商品体验, 物流体验, 服务体验)，满分100分。

### 商品体验分（最重要）
- **商品综合评分**：好评率 + 评价数量
  - 评价数<30条且销量不足 → 分数被"固定"
  - 解封方法：积累有效评价+增加销量
- **商品品质退货率**：因品质问题退货的比例
  - 高于行业均值会严重降权

### 物流体验分
- 48小时揽收率 ≥ 95%
- 物流单号空转会被处罚
- 618大促期间物流分"只涨不跌"

### 服务体验分
- 客服响应速度（目标<30秒）
- 售后处理时长（目标<12小时）
- 飞鸽会话不满意率
- 平台求助率

### 体验分对运营的影响
| 体验分 | 影响 |
|--------|------|
| ≥ 90 | 优质商家，流量倾斜 |
| 80-89 | 正常，基本不受限 |
| 70-79 | 部分功能受限，流量减少 |
| 65-69 | 严重受限，商品卡无法入池 |
| < 65 | 可能被清退 |

## 四、搜索排名机制

### 搜索流量爆发
- 抖音搜索日活1.5-2亿
- 月搜索量超200亿
- 搜索流量同比增长210%

### 搜索排名因素
1. **标题关键词匹配度**（权重最高）
2. **商品销量&评价**（有销量的排名靠前）
3. **店铺体验分**（高分优先）
4. **商品信息质量分**（满分100）
5. **点击率&转化率**（数据好的排名上升）

### 搜索优化方法
- 标题写满30字：品牌词+属性词+类目词+卖点词+长尾词
- 用"巨量算数"查热搜词
- 抖店后台 → 搜索运营 → 优化标题，看系统推荐词

## 五、流量分配逻辑

### 流量类型
| 类型 | 来源 | 占比趋势 |
|------|------|---------|
| 推荐流量 | 猜你喜欢/信息流 | 最大头 |
| 搜索流量 | 用户主动搜索 | 快速增长中 |
| 内容流量 | 短视频/直播挂车 | 转化率最高 |
| 付费流量 | 千川/DOU+ | 可控放量 |
| 活动流量 | 大促/秒杀 | 爆发性强 |

### 流量冷启动漏斗
新店/新品：前48小时完成全链路闭环（上架→曝光→下单→签收→评价）是入池关键。

## 六、权重体系

### 三重权重耦合
1. **单品权重**：销量+评价+转化率+点击率
2. **店铺权重**：体验分+全店动销率+违规记录
3. **达人权重**：带货达人的粉丝量+历史带货数据

### 核心要点
- 单品权重 ≠ 店铺权重（一个爆款不意味着全店有权重）
- 需80%以上在线商品产生真实交易（全店动销）
- 新品有冷启动扶持期，前48小时数据最重要
