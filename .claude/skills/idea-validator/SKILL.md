# 创意验证器

## 概述

在投入真金白银之前，通过系统研究、自我辩论、竞争分析和 AI 机会映射来验证一个商业想法。每个阶段由用户确认后再继续。

**写第一行代码前，先跑完这套流程。**

## 第一阶段：创意简报

问用户：
- 一句话说清楚这个想法是什么
- 目标客户是谁（具体是什么人）
- 解决什么痛点
- 目标客户现在怎么解决这个问题（现状方案）

给用户反馈一份结构化的简报。

**停下来。**
```
第一阶段完成。输入"确认"进入第二阶段，或补充修改意见。
```

## 第二阶段：细分市场 + 规模调研

用网页搜索调研市场，至少跑 5 次搜索：

**搜索关键词：**
- `"[细分领域] 市场规模 2024 2025"`
- `"[行业] 增长趋势 2025"`
- `"[痛点] 用户吐槽"` — 找真实用户痛感
- `"[痛点] 有没有解决方案"`
- `"[目标客户] 最大挑战"`

**需记录的内容：**
- 市场规模（TAM）- 标注来源
- 市场趋势（增长 / 稳定 / 萎缩）
- 用户反馈中 TOP 3 痛点
- 合规/政策相关风险
- 用户为什么还没解决这个问题

以清晰的摘要呈现调研结果，附来源。

**停下来。**
```
第二阶段完成。输入"确认"进入第三阶段，或补充修改意见。
```

## 第三阶段：自我辩论（正反方）

跟自己唱反调。诚实辩论。

**正方（为什么能成）：**
- 为什么这个事能做起来
- 创始人（你）有什么独特优势
- 时机——为什么现在是对的时候
- 什么大趋势（技术/行为/政策）在助攻

**反方（为什么可能失败）：**
- 12 个月内可能死在哪
- 隐藏的风险或盲区
- 如果做起来了，谁会抄
- 什么条件必须成立才能成功（这些条件成立吗？）
- 你很可能低估了什么

**最后给出：**
- 信心分：1-10（附诚实推理）
- 最大的单一风险

**停下来。**
```
第三阶段完成。输入"确认"进入第四阶段，或补充修改意见。
```

## 第四阶段：竞品扫描

用网页搜索找出谁已经在这个领域了。至少跑 6 次搜索：

**搜索关键词：**
- `"最好用的 [解决方案] 2025"`
- `"[解决方案] 推荐"`
- `"[痛点] 创业 产品"`
- `"[细分领域] 替代品"`
- `"[竞品名] 评价"` — 找到竞品后再搜

**对每个找到的竞品，记录：**

| 竞品 | 链接 | 定价 | 核心功能 | 弱点 |
|------|------|------|----------|------|
| | | | | |

**市场判断：**
- 这个市场是拥挤、新兴、还是空白？
- 没人做的 #1 空白是什么？
- 哪个竞品的不满用户最容易抢过来？

第四阶段完成。输入"确认"进入第五阶段，或补充修改意见。

## 第五阶段：AI + 自动化机会

识别 AI 或自动化能否对现有方案形成降维打击：

**需要回答的问题：**
- 这个领域哪些人工工作可以完全自动化？
- LLM 能在哪里带来 10 倍价值，而竞品不改架构就做不到？
- 能建立起什么数据护城河？
- 哪些功能只有在最近 AI 能力下才可能实现？
- AI 能否降低获客成本、提高留存、或改进核心产品效果？

**AI 机会评级：**
- 低 — AI 只是锦上添花
- 中 — AI 能明显改善产品
- 高 — AI 是核心差异化
- 变革级 — 没有 AI 产品不成立，竞品很难追上

## 第六阶段：Go / No-Go 最终判定

综合所有信息，给出打分：

| 维度 | 评分（1-5） | 说明 |
|------|------------|------|
| 市场规模 | | |
| 痛点强度 | | |
| 竞争空白 | | |
| AI/自动化机会 | | |
| 时机 | | |
| 创始人匹配度（如已知） | | |
| **总分** | **/30** | |

**分数解读：**
- 24-30：强力 GO — 可以放心推进
- 16-23：有条件 GO — 先解决特定风险再动
- 10-15：PIVOT — 信号对但形态或市场不对，调整方向
- 10 以下：NO-GO — 想法有根本性问题

**最终判定：** GO / 有条件 GO / PIVOT / NO-GO

如果是 PIVOT — 基于调研给出具体的转型建议。
如果是 GO — 建议下一步跑 `/competitor-analysis` 做深入竞品分析。

## 搜索规则

- 始终注明来源（URL 或"来自小红书/知乎/电商评论等"）
- 第二、第四阶段至少共 8-10 次搜索
- 如果一次搜索没结果，换同义词或更宽泛的关键词
- 优先使用 2024-2025 年的数据，避免过时信息
