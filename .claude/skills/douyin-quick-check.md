# Skill: douyin-quick-check

## 触发条件

当用户想快速了解店铺今日运营概况，不需要完整报告时使用。适合早晨到店后快速扫一眼。

## 功能

轻量级快速检查：只拉取核心指标卡数据，不翻页不提取完整表格，不做详细分析。

## 与 daily-analysis 的区别

| | quick-check | daily-analysis |
|---|---|---|
| 耗时 | ~2 分钟 | ~15 分钟 |
| 数据量 | 只看概览页 | 全部数据 + 翻页 |
| 输出 | 几行核心指标 | 完整日报 |
| 是否翻页 | 否 | 是 |
| 是否诊断 | 否 | 是 |

## 步骤

### Step 1: 快速登录验证

```
browser_navigate → https://fxg.jinritemai.com/
```

如果跳转到登录页 → 提示用户先执行 `douyin-login`。

### Step 2: 查看首页概览

抖店后台首页通常有数据概览卡片，展示核心指标。

1. `browser_snapshot` 获取页面结构
2. 提取以下指标卡：
   - 今日成交额
   - 今日订单数
   - 今日访客数
   - 退款中笔数
   - 店铺评分

### Step 3: 查看数据罗盘概览

```
browser_navigate → https://fxg.jinritemai.com/data/overview
```

提取近 7 天 GMV 走势（通常有折线图旁的数值）

### Step 4: 快速输出

```
📊 抖店速览 — YYYY-MM-DD HH:MM

💰 今日成交额: ¥X,XXX.XX
📦 今日订单数: XX 单
👁️  今日访客数: X,XXX
🔄 退款中: X 笔
⭐ 店铺评分: X.X

📈 近 7 天 GMV: ¥XX,XXX (较前 7 天 ↑/↓ XX%)

⚠️  需要关注: [如有告警，这里列出]
```

## 输出

纯文本，不生成文件。只是让用户快速了解今天店铺状态。

## 安全规则

- ✅ 只查看概览数据，不点击任何操作按钮
- ❌ 不修改任何店铺数据
