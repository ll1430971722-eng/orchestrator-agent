# 审片清单 — 公司号第一条互动视频 v3（真实素材+口播版）

## 内容审查
- [ ] 1. 是否使用了真实样品间素材（DJI 实拍，非 AI 生成）
- [ ] 2. 样品间画面是否确实是用户的文具/笔袋公司样品间
- [ ] 3. 口播内容是否自然（不过度浮夸、不播音腔）
- [ ] 4. 是否不是产品广告（没有推销感、没有产品特写主导画面）
- [ ] 5. 是否不是招商片（没有招商信息）
- [ ] 6. 是否有明确评论引导（字幕清晰列出选项）
- [ ] 7. 是否适合抖音竖屏观看（9:16）
- [ ] 8. 字幕是否清晰可读（白色字+黑边，位置合理）
- [ ] 9. 口播 + 字幕是否配合流畅
- [ ] 10. 是否比 v2 AI 版本更符合真实样品间 + 口播需求

## 技术审查
- [ ] 分辨率：720×1280（竖屏 9:16）
- [ ] 时长：8.6 秒
- [ ] 音频：有口播配音
- [ ] 无品牌 logo
- [ ] 无价格/促销/招商信息

## v2 vs v3 对比
| 维度 | v2 (Seedance) | v3 (真实素材+口播) |
|------|--------------|-------------------|
| 样品间来源 | AI 生成，不像真实样品间 | DJI 实拍，100% 真实 ✅ |
| 口播 | 无 | 有 AI 口播配音 ✅ |
| 时长 | 5s（冻结帧延至 8s） | 8.6s 自然时长 ✅ |
| 字幕 | 无 | Pillow + ffmpeg 叠加 ✅ |
| 分辨率 | 720p | 720p |

## 改进建议（如有）
- 如需更高分辨率，可重新用 DJI 原片 4K 导出
- 如需真人出镜，建议补拍员工在样品间讨论的画面
- 口播如需更自然，可替换为真人录音
