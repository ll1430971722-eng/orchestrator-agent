# 发布前审核清单（样品间参考版）

## 内容审核
- [ ] 前 1 秒是否有强钩子（笔袋样品举起/夸张表情/快速切入）
- [ ] 8 秒内是否能完整看完（节奏是否够快）
- [ ] 是否有明确评论引导（结尾停留给观众反应时间）
- [ ] 是否提供了评论选项（A-F 选项可见）
- [ ] 是否不像广告片（无产品特写推售、无logo、无品牌色）
- [ ] 是否像文具公司真实样品间（非普通办公室、非样板间）
- [ ] 是否有反差感（文具公司认真开会 vs 讨论抖音发什么）
- [ ] 样品架上产品是否为环境背景（非刻意产品展示）

## 画面审核
- [ ] 画面是否干净明亮
- [ ] 下方是否留出字幕空间（下1/3区域无关键视觉信息）
- [ ] 是否无清晰logo/品牌标识
- [ ] 是否无真实公司名
- [ ] 是否无清晰可读文字/数字
- [ ] 产品是否作为环境元素自然出现（非硬广）
- [ ] 样品架/笔袋/文具是否色彩丰富但不杂乱

## 与上一版对比
- [ ] 是否比通用办公室版更贴近文具公司真实场景
- [ ] 样品间元素是否自然融入画面（非后期合成感）

## 发布准备
- [ ] title 已选定
- [ ] caption 已选定
- [ ] hashtags 已确认
- [ ] 封面文案已确定
- [ ] 置顶评论已准备好
- [ ] 评论区选项已整理
- [ ] 发布后 30 分钟内有人值班回复评论

## 状态
- [ ] 本人已审阅，可以发布
