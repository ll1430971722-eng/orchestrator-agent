# 第一版重点指标

## 基础指标

| 指标 | 说明 | 关注点 |
|------|------|--------|
| GMV | 成交总额 | 涨跌趋势、环比变化 |
| 订单数 | 成交订单数 | 真实成交量，不含退款 |
| 退款金额 | 已退款总额 | 实际损失金额 |
| 退款率 | 退款订单/总订单 | 售后健康度 |
| 商品成交排行 | 各商品 GMV 排名 | 主力商品是谁 |
| 商品退款排行 | 各商品退款排名 | 问题商品是谁 |
| 商品库存风险 | 临近售罄或积压 | 库存健康度 |
| 售后异常 | 退款原因集中度 | 批量问题预警 |
| 近 7 天趋势 | 各项指标 7 日走势 | 判断是突发还是趋势 |

## 流量指标

| 指标 | 说明 | 关注点 |
|------|------|--------|
| 曝光 | 商品/内容被展示次数 | 平台给了多少"机会" |
| 点击率 | 点击/曝光 | 商品吸引力 |
| 访问量 | 进入店铺/商品页人数 | 实际进店流量 |
| 流量来源 | 各渠道流量占比 | 渠道健康度 |
| 搜索流量 | 用户搜索关键词进店 | 搜索排名情况 |
| 推荐流量 | 系统推荐进店 | 推荐权重情况 |
| 短视频流量 | 短视频带来的流量 | 内容引流效果 |
| 直播流量 | 直播带来的流量 | 直播引流效果 |
| 商城流量 | 商城自然流量 | 商城权重情况 |

## 转化指标

| 指标 | 说明 | 关注点 |
|------|------|--------|
| 加购率 | 加购人数/访问人数 | 商品兴趣度 |
| 支付转化率 | 支付人数/访问人数 | 成交效率 |
| 客单价 | GMV/订单数 | 用户消费水平 |
| 下单人数 | 创建订单的人数 | 下单意愿 |
| 支付人数 | 实际支付的人数 | 真实成交人数 |

## 广告指标

| 指标 | 说明 | 关注点 |
|------|------|--------|
| 广告消耗 | 广告花费总额 | 成本控制 |
| 点击率 | 广告点击/曝光 | 广告素材吸引力 |
| 转化成本 | 消耗/成交数 | 获客成本 |
| 成交金额 | 广告带来的 GMV | 广告贡献 |
| ROI | 成交金额/消耗 | 投入产出比 |
| 素材表现 | 各素材点击率和转化 | 素材质量 |
| 计划表现 | 各计划 ROI 和消耗 | 计划效率 |

## 售后指标

| 指标 | 说明 | 关注点 |
|------|------|--------|
| 退款率 | 退款订单占比 | 售后健康度 |
| 退款金额 | 退款金额 | 实际损失 |
| 退款原因 | 退款原因分布 | 问题定位 |
| 差评 | 差评数量和内容 | 用户不满来源 |
| 物流问题 | 物流相关投诉 | 发货体验 |
| 客服问题 | 客服相关投诉 | 服务体验 |

## 指标异常判断

### GMV 异常
- 日环比下降 > 30%：⚠️ 高风险
- 日环比下降 15%-30%：⚠️ 中等风险
- 日环比下降 < 15%：⚠️ 关注观察

### 退款率异常
- 退款率 > 行业均值 × 1.5：⚠️ 高风险
- 退款率连续 3 天上涨：⚠️ 需要排查

### 转化率异常
- 支付转化率日环比下降 > 20%：⚠️ 需要排查
- 加购率高但支付率低：⚠️ 价格或信任问题

### 广告 ROI 异常
- ROI < 1：⚠️ 亏损，需要立即关注
- ROI 连续 3 天下降：⚠️ 需要排查原因
- 单计划消耗高但 ROI 低：⚠️ 需要暂停观察

### 流量异常
- 曝光日环比下降 > 30%：⚠️ 需要排查渠道
- 点击率日环比下降 > 20%：⚠️ 主图或标题问题
- 搜索流量大幅下降：⚠️ 搜索排名变化

## 数据不足时的处理

当某个维度的数据不可用时：
1. 在报告中标注"数据缺失"
2. 说明缺少哪个数据源
3. 说明缺失对分析结论的影响
4. 给出补充数据的建议
5. 不要基于不完整的数据下确定性结论
